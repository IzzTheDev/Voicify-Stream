# VoicifyStream Desktop 🎙️
## Advanced Text-to-Speech App for Windows

A standalone desktop application for realistic text-to-speech generation

![VoicifyStream](screenshot.png)

---

## Features

- ✅ **16 Realistic Voices**: Guy, Jenny, Aria, Andrew, Brian, Emma, Ava, Ryan, Sonia, Libby, Natasha, William, Clara, Liam, Neerja, Prabhat
- ✅ **Speed Control**: -50% to +100%
- ✅ **Play/Stop**: Full playback controls
- ✅ **Save Audio**: Export to any location
- ✅ **Dark Theme**: Easy on the eyes!

---

## System Requirements

- 1920 x 1080 display
- Nvidia Geforce GT 1080
- AMD Ryzen 2000
- 1GB ram
